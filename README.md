# xTag

Automatic version updater

## Install
You can install `xTag` using 

### curl

```
curl -L https://raw.githubusercontent.com/kaunteya/xTag/master/installer.sh | sudo sh
```

### Manual
```
make install
```

## Usage
Run this command in your repo
```
xtag Info.plist p
```
